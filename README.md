# VueNote简介
基于Vue.js + .NetCore的在线笔记应用，采用前后端分离架构。  
功能、UI设计上参考了印象笔记的Web版。 

# 技术栈
1. 前端：Vue.js 2.5、Vue-Router、Axios、ElementUI
2. 服务端：ASP.NET Core 2.2（MVC)、Dapper
3. 数据库：MySql 8.0
4. 操作系统：Ubuntu 18.10
4. 开发工具：Visual Studio 2017、VSCode
