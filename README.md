# VueNote简介
基于Vue.js + .NetCore的在线笔记应用，采用前后端分离架构。  
功能、UI设计上参考了印象笔记的Web版。  

# 技术栈
1. 前端：vue、vue-router、axios、elementUI
2. 服务端：asp.netcore mvc、dapper
3. 数据库：mysql
